# Claude Code + React 学习任务清单（跳槽实战版）

> 目标：通过一个真实 React 项目，在 3–4 个月内同时掌握 Claude Code 与 React 工程能力，为前端跳槽做好准备。

---

# 一、项目学习目标

这个学习项目有三个核心目标：

1. 系统掌握 Claude Code 的工作方式
2. 通过 React 项目练习真实开发能力
3. 为跳槽准备可以展示的项目经验

最终你需要具备三种能力：

- 使用 Claude Code 探索和理解代码库
- 使用 Claude Code 辅助开发 React 项目
- 能够解释你的设计思路并通过面试

---

# 二、学习原则

本项目遵循以下原则：

- 先理解再写代码
- 先小功能再复杂系统
- Claude 是老师而不是代码生成器
- 每个阶段必须有练习任务
- 每个阶段必须有复盘总结

---

# 三、学习路线总览

学习过程分为九个阶段：

阶段 0：环境准备  
阶段 1：Claude Code 基础工具  
阶段 2：Claude Code 常用命令  
阶段 3：项目探索能力  
阶段 4：React 项目开发  
阶段 5：调试与优化  
阶段 6：Git 与代码管理  
阶段 7：Claude Code 记忆系统  
阶段 8：高级能力（Subagent / Skills）  
阶段 9：项目复盘与面试表达

---

# 四、阶段 0：环境准备

目标：完成 Claude Code 学习环境搭建。

任务：

- 配置全局 `~/.claude/CLAUDE.md`
- 配置项目级 `CLAUDE.md`
- 确认 Claude Code 可以正常运行
- 确认当前模型和 effort 设置

练习：

输入以下命令：

/help  
/status  
/context  

然后在项目中输入：

带我理解这个项目结构，不要直接改代码

验证标准：

- Claude 可以解释项目结构
- Claude 能按教学方式回应

---

# 五、阶段 1：Claude Code 基础工具

这一阶段重点学习 Claude Code 的基础工具。

---

## 1 Glob（文件查找）

作用：

通过模式查找文件。

练习：

- 查找所有 `.tsx` 文件
- 查找所有组件文件
- 查找所有样式文件
- 查找所有配置文件

示例问题：

查找项目中所有 React 组件文件  
查找所有 `.ts` 文件

验证：

你可以快速定位组件和页面文件。

---

## 2 Grep（内容搜索）

作用：

在代码中搜索某个关键字。

练习：

搜索：

- useState
- useEffect
- console.log
- API 请求
- 路由定义

示例：

搜索所有 useEffect 的使用位置

验证：

你能找到某个功能在哪里实现。

---

## 3 Read（读取文件）

作用：

让 Claude 精读关键文件。

练习：

读取：

- package.json
- 项目入口文件
- App 组件
- 路由配置
- README

示例：

读取 package.json 并解释项目技术栈。

验证：

你知道项目如何启动。

---

## 4 Write（创建文件）

作用：

创建新文件。

练习：

创建：

- Button 组件
- TodoItem 组件
- useTodos hook
- utils 工具文件

原则：

只创建最小骨架。

---

## 5 Edit（修改文件）

作用：

对现有代码做最小改动。

练习：

- 给组件增加 props
- 添加删除功能
- 添加 loading 状态
- 添加表单输入

原则：

最小修改。

---

## 6 Bash（执行命令）

作用：

执行系统命令。

练习：

- 安装依赖
- 启动项目
- 执行构建
- 查看目录
- 查看 git 状态

验证：

你知道什么时候读代码，什么时候执行命令。
